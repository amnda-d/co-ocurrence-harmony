# A universal tendency towards vowel harmony and consonant anti-harmony

- Python 3.10.0
- R 4.0.3
  - brms 2.16.3
  - tidybayes 3.0.1
  - bayestestR 0.11.0
  - tidyverse 1.3.1
  - cowplot 1.1.1

```
make run
Rscript r/models/script.R {datafilename.csv}
```
